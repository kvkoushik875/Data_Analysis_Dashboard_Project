# **Interactive Excel Dashboard for Hotel Booking Data Analysis**
**Project Objective:**
To analyze hotel booking and cancellation data using Microsoft Excel in order to identify booking patterns, cancellation trends, and key factors influencing cancellations, and to present actionable insights through an interactive dashboard for data-driven decision-making.
## Dataset Used
- <a href="https://github.com/kvkoushik875/Data_Analysis_Dashboard_Project/blob/main/Hotel_booking_cancellation_dashboard.xlsx">Dataset</a>

## Key Questions Answered by the Dashboard
- What is the overall booking vs cancellation rate?
- Which hotel type (City Hotel vs Resort Hotel) has a higher cancellation rate?
- How does lead time affect the likelihood of cancellation?
- Which customer segments (Transient, Group, Contract) cancel the most?
- What is the impact of deposit type on booking cancellations?
- Which market segments contribute most to total bookings and cancellations?
- How do monthly and seasonal trends influence booking demand and cancellations?
- Which room types experience the highest number of cancellations?
- How do repeated guests vs new guests differ in cancellation behavior?
- What are the top factors driving cancellations, and where can improvements be made?

- Dashboard Interactions <a href="https://github.com/kvkoushik875/Data_Analysis_Dashboard_Project/blob/main/Dashboard-Data%20Analysis%20Project.png">Dashboard</a>

# Process 
- Collected and reviewed raw hotel booking data in Excel.
- Cleaned data by handling missing values, duplicates, and inconsistencies.
- Created calculated fields and KPIs using Excel formulas.
- Built pivot tables and pivot charts for analysis.
- Designed an interactive dashboard using slicers and formatting.
